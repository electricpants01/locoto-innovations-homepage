# 🤖 AI Context Folder

This folder contains persistent memory and context for AI assistants working on this project.
**Read these files at the start of every session** to instantly understand the project, the owner, and what has been done.

---

## 📂 Folder Structure

```
AI/
├── README.md           ← You are here — start here every session
├── profile.md          ← Christian's full profile, bio, work experience, social links
├── architecture.md     ← Tech stack, design system, file structure, Astro/Tailwind config
├── context.md          ← Current project state: what's done, what's pending, decisions made
│
├── skills/
│   ├── README.md       ← Overview of all skill categories
│   ├── mobile.md       ← Mobile development skills (Kotlin, Android, React Native, Swift)
│   ├── architecture.md ← Software architecture patterns (MVVM, Clean Arch, Coroutines)
│   └── tools.md        ← Libraries & tools (Firebase, Room, Retrofit, Fastlane, Git)
│
└── agents/
    ├── README.md        ← Overview of all agent roles
    ├── ui-agent.md      ← UI Agent: design, styling, Tailwind, responsive layout
    ├── content-agent.md ← Content Agent: bio, experience, projects, blog posts
    ├── dev-agent.md     ← Dev Agent: new pages, components, integrations, architecture
    ├── blog-agent.md    ← Blog Agent: Medium integration, RSS, newsletter
    └── deploy-agent.md  ← Deploy Agent: Vercel/Netlify, CI/CD, custom domain
```

---

## 🚀 Quick Start for AI Sessions

1. **Read `profile.md`** → know who the owner is and their full background
2. **Read `architecture.md`** → understand the tech stack and project structure
3. **Read `context.md`** → know exactly what state the project is in
4. **Browse `skills/`** → deep dive into any skill category
5. **Check `agents/`** → pick the right agent role for the task at hand
